# 剪贴板助手

一个基于 Electron 的 Windows 桌面剪贴板工具，支持文本、图片、递归粘贴队列、固定位置插入、文件导入、组记录、托盘运行和快捷键唤起。

[English README](./README.md)

## 功能特性

- 支持保存文本和图片剪贴板历史。
- 历史记录数量可在设置中配置，范围为 `10-1000`。
- 支持把历史项指定固定到某个粘贴位置。
- 支持多行复制：每一行作为一个粘贴项。
- 支持导入 `TXT`、`CSV`、`Excel` 文件。
- 支持组记录：保存常用的一组文本，并一键应用为粘贴队列。
- 支持监听 `Ctrl+V`，粘贴后自动准备下一项。
- 支持最小化到托盘、关闭隐藏到托盘、开机自启。
- 支持设置全局快捷键，快速打开工具窗口。
- 支持安装包和 Portable 免安装版打包。

## 使用方式

### 剪贴板

复制文本或图片后，内容会进入历史记录。

每条历史记录右侧的 `...` 菜单支持：

- `删除`
- `指定位置固定`
- `取消固定位置 x`

点击历史项本身会把该项重新复制到系统剪贴板。

### 多行复制

点击 `多行复制`，输入多行文本，例如：

```text
网址
用户名
密码
```

提交后会生成一个递归粘贴队列，并先复制第一项。之后在其它软件里正常按 `Ctrl+V` 粘贴，应用会自动准备下一项。

### 文件导入

点击 `导入` 后选择文件：

- `TXT`：按行读取。
- `CSV`：按单元格读取，忽略空单元格。
- `Excel`：只读取第一个 Sheet，按单元格读取，忽略空单元格。

导入历史记录只显示文件绝对路径，不显示文件内容。

### 固定位置规则

固定位置表示该固定项会插入到指定的粘贴顺序位置。

普通复制场景：

- 只生效固定位置 `2`。
- 例如复制 `B`，并且 `A` 固定在位置 `2`，粘贴顺序为：

```text
B -> A
```

多行复制 / TXT 导入：

如果只有 `G` 固定在位置 `2`，普通项是：

```text
A
B
C
D
```

粘贴顺序为：

```text
A -> G -> B -> G -> C -> G -> D
```

例如普通项是：

```text
A
B
C
D
```

固定位置为 `2`、`4`、`5`、`10`，粘贴顺序为：

```text
A -> 固定位置2 -> B -> 固定位置4 -> 固定位置5 -> C -> D
```

位置 `10` 没有用到，因此不会插入。

CSV / Excel 导入：

每一行看作一次多行复制，每一行都从头应用固定位置规则。

### 组记录

`组记录` 页面支持：

- 搜索
- 新增
- 应用
- 编辑
- 删除

新增或编辑组记录时：

- `name` 不能重复。
- 多行文本框中每一行都是一个 item。
- 删除组记录前会二次确认。

点击 `应用` 后，组记录会按多行复制规则进入粘贴队列。

### 设置

设置页包含：

- 最小化到托盘
- 关闭时隐藏到托盘
- 开机自启
- 粘贴后自动推进
- 历史记录数量
- 快捷打开工具

`快捷打开工具` 默认为空。点击输入框后按下组合键，例如 `Ctrl + Alt + K`，保存后即可用该快捷键快速打开窗口。

## 开发

安装依赖：

```powershell
npm install
```

启动开发版：

```powershell
npm start
```

运行语法检查：

```powershell
node --check main.js
node --check preload.js
node --check renderer\app.js
```

## 打包

同时生成安装包和 Portable：

```powershell
npm run dist
```

只生成安装包：

```powershell
npm run dist:installer
```

只生成 Portable：

```powershell
npm run dist:portable
```

输出目录：

```text
release/
```

常见输出：

```text
剪贴板助手-0.1.0-安装包.exe
剪贴板助手-0.1.0-Portable.exe
```

## 依赖

- Electron
- electron-builder
- xlsx
- uiohook-napi

普通用户运行已打包的安装包或 Portable 文件时，不需要安装 Node.js 或 npm。

## 注意事项

- Windows 剪贴板本身不会通知应用“内容已经被粘贴”。本工具通过监听 `Ctrl+V` 来判断粘贴动作，并在粘贴后准备下一项。
- 右键菜单粘贴、软件内置粘贴按钮或特殊粘贴方式不一定能触发自动推进。
- 固定位置项可以是文本或图片。
- 组记录仅支持文本。
